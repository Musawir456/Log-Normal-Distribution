## 🧠 What is Log-Normal Distribution?

A variable is **Log-Normally distributed** if:

```
If Y = ln(X) is normally distributed,
Then X is log-normally distributed.
```

In simple words:

- Normal distribution → symmetric
- Log-normal distribution → right-skewed (positively skewed)

---

## 📌 Key Characteristics

- Always positive values (X > 0)
- Right-skewed distribution
- Mean > Median > Mode
- Used when data grows multiplicatively

## 📊 Mathematical Formula

If:

```
Y ~ N(μ, σ²)
```

Then:

```
X = e^Y
```

Where:
- μ = Mean of log values
- σ = Standard deviation
- e = Euler’s number
